# Lumio — STR Intelligence

AI-native short-term rental analytics platform. POC built with real Airbnb data.

## Quick start

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to vercel.com → New Project → Import from GitHub
3. Click Deploy — done

## Stack

- React 18 + Vite
- Recharts for data visualization
- Real Airbnb host data (Bassano del Grappa, Veneto)
